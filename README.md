2. React.js
3. Redux thunk
4. Typescript
5. MUI Antd
6. Axios

8. React testing-library Jest

2. Lombok & Mapstruct
3. Flyway & PostgreSQL (JPA)
4. Swagger & Postman
5. Spring Security & JWT
6. Unit Test & Jacoco & Mockito
7. Gradle
8. RabbitMQ (amqp)
9. Amazon S3 (MinIO)
10. Redis (Jedis)
11. Sockjs (websocket)
12. Sendgrid (Email)
