# ThinkMoreForum-Backend

## Requirements

* Openjdk11  
* Gradle
* PostgreSQL
* MinIO
* RabbitMQ

## How to Build for Production
```shell
$ chmod +x gradlew
$ ./gradlew build
```

## How to Develop in Local Environment
```shell
$ docker-compose up -d
```
Then use IDE idea open the project and run ForumApplication
